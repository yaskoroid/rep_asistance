Текст ТЗ:
Создать приложение для Android.

1. Пользователь видит список сотрудников, информация: фамилия, инициалы (по имени), возраст, город
2. Пользователь нажимает на сотрудника из списка, открывается экран редактирования. Поля:
  - имя
  - фамилия
  - год рождения
  - город (на выбор Киев, Харьков, Одесса, Львов)
  - должность (на выбор врач, фармацевт, медсестра, водитель)
3. Пользователь может сохранить изменения.
4. На первом экране должна присутствовать кнопка добавления нового сотрудника, поля те же.

Код приложения разместите в публичном репозитории (github, bitbucket, и пр), чтобы его можно было получить и собрать.

Критерии
--------

1. Соответствие спецификации.
2. Удобство UI.
3. Качество кода (простота, читаемость).
